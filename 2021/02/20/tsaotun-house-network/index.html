有線網路與第四台配線紀錄

發表於 2021-02-20 閱讀次數： Disqus：

老家最近正在整修，趁過年有空，幫忙整理網路線與第四台的線路，有以下幾個目標需要達成：

  1. 未來電視要能連接wifi觀看串流影音
  2. 頻寬必須滿足未來使用NAS的需求
  3. 無線網路必須滿足IP cam監視器的需求
  4. 架設 mesh wifi，提升收訊品質

線路規劃

參考最上方的示意圖，老家為兩棟相鄰的透天厝，右半部目前正在整修，家人主要起居都位於左半部，故於左半部的、二樓各放置一台mesh wifi AP。右半的二樓有兩條明線，來自左半的網路訊號需經過SW2的橋接，再送到右下的一樓。

材料列表

  • 剝線鉗x1
  • 卡式資訊座x4
  • 同軸插座x2
  • RJ-45接頭x1包
  • 壓線鉗x1
  • Cat-5e線材20m+
  • 雙孔面板x3
  • 拉線器30m
  • 插座螺絲x6

拉線與安裝插座

由較空曠的右半部開始，先將穿線器由三樓穿至一樓，將網路線與同軸線綁上後再拉至三樓。由於網路線夠長，直接由三樓的左半往下穿至一樓。同軸線較短無法直接拉到左半一樓，因此再用另一條由三樓往下穿至一樓，再將三樓的兩條同軸線接在一起。

線拉好後就可以安裝插座，同軸線比較麻煩，買到的接頭是旋入式的鳳梨頭，必須調整好剝線長度才能安裝。網路插座相對單純，將最外層的皮剝掉後，八條細線按照顏色壓進插座即可，不用剝細線的皮。壓線可以利用剝線鉗後端的部分，必須要壓到最底才能確保線路穩固。
